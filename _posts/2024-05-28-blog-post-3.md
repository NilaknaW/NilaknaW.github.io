This is a simulation project for visualizing the approximation to Pi by counting collisions between boxes. 
For the 3d physics simulation, I used Unity3D and the project is inspired by the video posted on the 3Blue1Brown YouTube channel. 
The simulation was made for the mathematics department stall at the EXMO 2023 engineering exhibition of the University of Moratuwa.

Read the pdf to learn about the math behind the project!

To run the simulation yourself, download the simulation folder and run 'math 2 1.exe' in github repo.
[https://github.com/NilaknaW/Approximating-Pi-with-colliding-boxes-Unity-simulation](https://github.com/NilaknaW/Approximating-Pi-with-colliding-boxes-Unity-simulation)
