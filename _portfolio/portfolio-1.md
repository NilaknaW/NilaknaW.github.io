This project aimed at developing a computer vision-based solution for recognizing Sri Lankan license plates in real-time video from an IP camera. Python and OpenCV library are utilized in processing video inputs. The Ultralytics YOLO model is used for vehicle and license plate detection after training on a custom dataset from Roboflow. Tracking vehicles is done using the SORT algorithm while reading the content on the license plate is done using the EasyOCR module. Flask, HTML, and CSS are used to visualize the results on a web page.
